在网页中使用Audio API - 生成指定频率的波形

狐妖i 前端Web Audio API

Oscillator 节点可以用于生成指定频率的某一种波形(例如正弦波或者方波什么的)

参数

type 类型是字符串

决定 OscillatorNode 播放的声音的周期波形,不同的波形可以产生不同的声调. 基础值有 “sine”(正弦波),“square”(方波),“sawtooth”(锯齿波), “triangle”(三角波) 与 “custom”(自定义). 默认值是”sine”(正弦波)。

frequency.value 类型是整数

代表了声音的振动频率(单位为赫兹). 默认值是 440 Hz (中A音高).

frequency.value 类型是整数

代表了音高微调(单位是音分),默认值是0.

演示 在新页面打开 (没有声音?请在网站设置中将该网站声音权限设置为允许)
